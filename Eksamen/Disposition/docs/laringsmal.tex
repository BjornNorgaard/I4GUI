\section{Læringsmål}

\begin{itemize}
	\item Redegøre for principperne i .Net frameworket og dets overordnede arkitektur samt beskrive og anvende programmeringssproget C\#.
	\item Designe og implementere programmer med en grafisk brugergrænseflade til Microsoft Windows platformen med brug af .Net frameworket og programmeringssprogene C\# og XAML.
	\item Anvende kontroller til opbygning af både modale og modeless dialoger, samt kunne anvende de forskellige layout panels.
	\item Anvende WPF's faciliteter til tegning af 2D grafik samt visning af billeder.
	\item Anvende styles og ressourcer.
	\item Anvende .Net frameworkets faciliteter til persistering af applikation- og brugerindstillinger samt til persistering af data i filer.
	\item Anvende data binding til at sammenknytte data i modellaget med deres præsentation i viewlaget.
	\item Redegøre for WPF's faciliteter til kommunikation mellem bagrundstråde og GUI-tråden i flertrådede programmer.
	\item Redegøre for arkitekturen for en Webapplikation.
	\item Designe og implementere Webapplikationer med en grafisk brugergrænseflade med brug af HTML5, CSS og javascript.
	\item Anvende et server side MVC framework til udvikling af Webapplikationer.
\end{itemize}
