\section{Læringsmål}

\begin{itemize}
	\item [\ref{sec:spm1}] Redegøre for principperne i .Net frameworket og dets overordnede arkitektur samt beskrive og anvende programmeringssproget C\#.
	\item [\ref{sec:spm2}] Designe og implementere programmer med en grafisk brugergrænseflade til Microsoft Windows platformen med brug af .Net frameworket og programmeringssprogene C\# og XAML.
	\item [\ref{sec:spm3}] Anvende kontroller til opbygning af både modale og modeless dialoger, samt kunne anvende de forskellige layout panels.
	\item [\ref{sec:spm4}] Anvende WPF's faciliteter til tegning af 2D grafik samt visning af billeder.
	\item [\ref{sec:spm5}] Anvende styles og ressourcer.
	\item [\ref{sec:spm6}] Anvende .Net frameworkets faciliteter til persistering af applikation- og brugerindstillinger samt til persistering af data i filer.
	\item [\ref{sec:spm7}] Anvende data binding til at sammenknytte data i modellaget med deres præsentation i viewlaget.
	\item [\ref{sec:spm8}] Redegøre for WPF's faciliteter til kommunikation mellem bagrundstråde og GUI-tråden i flertrådede programmer.
	\item [\ref{sec:spm9}] Redegøre for arkitekturen for en Webapplikation.
	\item [\ref{sec:spm10}] Designe og implementere Webapplikationer med en grafisk brugergrænseflade med brug af HTML5, CSS og javascript.
	\item [\ref{sec:spm11}] Anvende et server side MVC framework til udvikling af Webapplikationer.
\end{itemize}
